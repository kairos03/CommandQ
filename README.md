# CommandQ
CommandQ is shell command management system.  
Command run the command in the queue sequentially.

# Quick start
1. start server
```
cq server start
```

2. add command
```
cq add [command]
```

3. show queued command list
```
cq ls
```

4. delete or kill command
```
cq del [command No.]
```

5. stop server
```
cq server stop
```
